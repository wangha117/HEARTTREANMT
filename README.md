# HEARTTREANMT
# 热处理工艺优化系统

## 功能特性
- 基于机器学习的硬度预测
- 三维可视化工艺参数分析
- Web端交互界面

## 快速启动
```bash
git clone https://github.com/wangha/HEARTTREANMT.git
cd HEARTTREANMT/app
pip install -r requirements.txt
python app.py
项目结构
app/
├── templates/    # 网页模板
├── static/       # 静态资源
├── model/        # 训练模型
└── app.py        # Flask主程序
